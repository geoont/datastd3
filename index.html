datastd3

The repository contains dataset for the three topographic map standards from the United States (SDTS), Taiwan (TTMDS), and Russian Federation (RFK). It serves as a supplementary material for a work comparing topographic and hydrographic categories between data standards from which the major languages used are significantly different. Each dataset specifies the formalized semantics of the categories and organizing structures of these categories within a data standard, which provides the basis for understanding the mismatches and compatibilities between similar categories from the three data standards, and thus the languages used in the three countries.

For more information an acces data file visit project repository at https://github.com/geoont/datastd3
